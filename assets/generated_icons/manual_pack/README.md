# Manual Icon Pack

This folder contains a cohesive alternate icon set for the app, kept separate from the current production assets.

Files:
- `app_mark.svg`
- `tasks.svg`
- `coupled.svg`
- `habits.svg`
- `notes.svg`
- `budget.svg`
- `reader.svg`
- `projects.svg`
- `settings.svg`
- `shopping.svg`

Style notes:
- Transparent background
- Shared teal / coral / navy / cream palette
- Navy outlines and saturated accents for visibility on both light and dark UI
- Simple high-contrast symbols for quick recognition
- SVG format so the icons can be scaled cleanly before any final PNG export

Suggested feature mapping:
- Tasks -> `tasks.svg`
- Coupled -> `coupled.svg`
- Habits -> `habits.svg`
- Notes -> `notes.svg`
- Budget -> `budget.svg`
- Reader -> `reader.svg`
- Projects -> `projects.svg`
- Settings -> `settings.svg`
- Shopping -> `shopping.svg`
- App icon concept -> `app_mark.svg`
